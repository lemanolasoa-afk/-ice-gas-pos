# Manual Test Checklist - Ice Gas POS

คู่มือทดสอบระบบแบบ Manual สำหรับทดสอบทุกฟีเจอร์

## ข้อมูลทดสอบ
- **URL**: https://localhost:5173/
- **Admin PIN**: 1234
- **พนักงาน PIN**: 0000

---

## 1. การเข้าสู่ระบบ (Login)

### 1.1 Login ด้วย Admin
- [ ] เปิดแอพ → แสดง Login Modal
- [ ] ใส่ PIN: 1234 → เข้าสู่ระบบสำเร็จ
- [ ] แสดงชื่อ "Admin" และ "ผู้ดูแลระบบ" ใน Header
- [ ] เห็นเมนูทั้งหมดใน Bottom Nav

### 1.2 Login ด้วยพนักงาน
- [ ] Logout → Login ด้วย PIN: 0000
- [ ] แสดงชื่อ "พนักงาน" และ "พนักงาน" ใน Header
- [ ] ไม่เห็นเมนูบางอย่าง (เช่น จัดการผู้ใช้)

### 1.3 Login ผิด
- [ ] ใส่ PIN ผิด → แสดง error "PIN ไม่ถูกต้อง"

---

## 2. หน้าขายสินค้า (POS)

### 2.1 แสดงสินค้า
- [ ] เห็นสินค้าแบ่งตามหมวดหมู่ (น้ำแข็ง, แก๊ส, น้ำดื่ม)
- [ ] แสดงชื่อ, ราคา, หน่วย
- [ ] สินค้าหมดแสดง "สินค้าหมด" และกดไม่ได้
- [ ] สินค้าใกล้หมดแสดง "เหลือ X"

### 2.2 เพิ่มสินค้าลงตะกร้า
- [ ] กดสินค้าน้ำแข็ง → เพิ่มลงตะกร้าทันที
- [ ] กดสินค้าน้ำดื่ม → เพิ่มลงตะกร้าทันที
- [ ] แสดงจำนวนบน card สินค้า
- [ ] ปุ่มตะกร้าด้านล่างแสดงจำนวนและยอดรวม

### 2.3 เพิ่มแก๊สลงตะกร้า
- [ ] กดสินค้าแก๊ส → แสดง Modal เลือกประเภท
- [ ] เลือก "แลกถัง" → ราคาปกติ
- [ ] เลือก "ค้างถัง" → ราคา + มัดจำ
- [ ] ไม่มีตัวเลือก "ซื้อขาด"

### 2.4 ค้นหาสินค้า
- [ ] พิมพ์ชื่อสินค้า → กรองสินค้าที่ตรง
- [ ] ลบคำค้นหา → แสดงสินค้าทั้งหมด

### 2.5 สแกนบาร์โค้ด
- [ ] กดปุ่มสแกน → เปิดกล้อง
- [ ] สแกนบาร์โค้ด → เพิ่มสินค้าลงตะกร้า

### 2.6 เปลี่ยนมุมมอง
- [ ] กดปุ่ม Grid → แสดงแบบ Grid
- [ ] กดปุ่ม List → แสดงแบบ List

---

## 3. หน้าตะกร้า (Cart)

### 3.1 แสดงรายการ
- [ ] แสดงสินค้าทั้งหมดในตะกร้า
- [ ] แสดงชื่อ, ราคา, จำนวน, รวม

### 3.2 แก้ไขจำนวน
- [ ] กด + → เพิ่มจำนวน
- [ ] กด - → ลดจำนวน
- [ ] กด - เมื่อเหลือ 1 → ลบสินค้า

### 3.3 เปลี่ยนประเภทแก๊ส
- [ ] สินค้าแก๊สแสดงปุ่ม "แลกถัง" และ "ค้างถัง"
- [ ] กดเปลี่ยน → ราคาเปลี่ยนตาม
- [ ] ไม่มีปุ่ม "ซื้อขาด"

### 3.4 ล้างตะกร้า
- [ ] กดปุ่มถังขยะ → ยืนยัน → ล้างตะกร้า

---

## 4. การชำระเงิน (Payment)

### 4.1 เปิด Modal ชำระเงิน
- [ ] กด "ชำระเงิน" → เปิด Modal
- [ ] ค่าเริ่มต้น "รับเงิน" = ยอดรวม (พอดี)
- [ ] สามารถกด "ยืนยันการชำระ" ได้ทันที

### 4.2 ชำระเงินสด
- [ ] เลือก "เงินสด" (default)
- [ ] ใส่จำนวนเงิน → แสดงเงินทอน
- [ ] กดปุ่มจำนวนเงินด่วน (100, 500, 1000)
- [ ] กด "ยืนยัน" → แสดง Success Modal พร้อมเงินทอน

### 4.3 ชำระโอนเงิน
- [ ] เลือก "โอนเงิน"
- [ ] แสดงยอดที่ต้องโอน
- [ ] กด "ยืนยัน" → สำเร็จ

### 4.4 วางบิล
- [ ] เลือก "วางบิล"
- [ ] ต้องเลือกลูกค้าก่อน
- [ ] ใส่หมายเหตุได้
- [ ] กด "ยืนยัน" → สำเร็จ

### 4.5 เลือกลูกค้า
- [ ] กด "เลือกลูกค้าสมาชิก" → เปิด Modal
- [ ] ค้นหาลูกค้า → แสดงผลลัพธ์
- [ ] เลือกลูกค้า → แสดงชื่อและแต้ม
- [ ] สามารถใช้แต้มแลกส่วนลดได้

### 4.6 เพิ่มลูกค้าใหม่
- [ ] กด "เพิ่มลูกค้าใหม่" → เปิด Form
- [ ] ใส่ชื่อ (บังคับ) และเบอร์โทร (ไม่บังคับ)
- [ ] กด "บันทึก" → เพิ่มและเลือกลูกค้าใหม่

### 4.7 ใช้โปรโมชั่น
- [ ] ถ้ามีโปรโมชั่น → แสดงปุ่ม "มีโปรโมชั่น X รายการ"
- [ ] กดเลือก → แสดงส่วนลด

### 4.8 Success Modal
- [ ] แสดง "สำเร็จ"
- [ ] แสดงเงินทอน (หรือ "พอดี" ถ้าไม่มี)
- [ ] กด "ตกลง" → กลับหน้าขาย

---

## 5. ประวัติการขาย (History)

### 5.1 แสดงรายการ
- [ ] แสดงรายการขายทั้งหมด
- [ ] แสดงวันที่, เวลา, ยอดรวม, จำนวนรายการ

### 5.2 ดูรายละเอียด
- [ ] กดรายการ → แสดงรายละเอียด
- [ ] แสดงสินค้า, ราคา, จำนวน
- [ ] แสดงวิธีชำระเงิน, ลูกค้า (ถ้ามี)

### 5.3 พิมพ์ใบเสร็จ
- [ ] กดปุ่มพิมพ์ → พิมพ์ใบเสร็จ

---

## 6. จัดการสินค้า (Products)

### 6.1 แสดงสินค้า
- [ ] แสดงสินค้าทั้งหมด
- [ ] แสดงสต็อก, ราคา, ต้นทุน

### 6.2 แก้ไขสินค้า
- [ ] กดสินค้า → เปิด Form แก้ไข
- [ ] แก้ไขข้อมูล → บันทึก

### 6.3 เพิ่มสินค้า
- [ ] กด "เพิ่มสินค้า" → เปิด Form
- [ ] ใส่ข้อมูล → บันทึก

---

## 7. รับสินค้าเข้าสต็อก (Stock Receipt)

### 7.1 รับสินค้า
- [ ] เลือกสินค้า
- [ ] ใส่จำนวน
- [ ] ใส่ต้นทุน (ไม่บังคับ)
- [ ] กด "บันทึก" → สต็อกเพิ่ม

---

## 8. ปิดยอดสต็อกน้ำแข็ง (Daily Stock Count)

### 8.1 บันทึกสต็อก
- [ ] แสดงสินค้าน้ำแข็งทั้งหมด
- [ ] แสดงสต็อกในระบบ, ขายวันนี้, คงเหลือควร
- [ ] ใส่สต็อกจริง → คำนวณการละลาย
- [ ] แสดง warning ถ้าละลายผิดปกติ
- [ ] กด "บันทึก" → อัพเดทสต็อก

---

## 9. รายงาน (Reports)

### 9.1 รายงานยอดขาย
- [ ] แสดงยอดขายตามช่วงเวลา
- [ ] กรองตามวัน/สัปดาห์/เดือน

### 9.2 รายงานกำไร
- [ ] แสดงกำไรตามช่วงเวลา
- [ ] แสดง margin %

### 9.3 รายงานสต็อก
- [ ] แสดงสต็อกทั้งหมด
- [ ] แสดงมูลค่าสต็อก

### 9.4 รายงานการละลาย
- [ ] แสดงการละลายตามช่วงเวลา
- [ ] แสดงกราฟแนวโน้ม

---

## 10. ตั้งค่า (Settings)

### 10.1 เมนูหมวดหมู่
- [ ] แสดงเมนูแบ่งหมวดหมู่ (สต็อก, รายงาน, ลูกค้า, ระบบ)
- [ ] กดหมวด → ขยาย/ยุบ

### 10.2 การแจ้งเตือน
- [ ] เปิด/ปิดการแจ้งเตือนได้

### 10.3 เครื่องพิมพ์
- [ ] เชื่อมต่อ/ยกเลิกเครื่องพิมพ์ Bluetooth

### 10.4 ออกจากระบบ
- [ ] กด "ออก" → ยืนยัน → กลับหน้า Login

---

## 11. Offline Mode

### 11.1 ทำงานแบบ Offline
- [ ] ปิด Internet → แสดงแถบ "ออฟไลน์"
- [ ] ยังขายสินค้าได้
- [ ] เปิด Internet → ซิงค์ข้อมูลอัตโนมัติ

---

## 12. PWA

### 12.1 ติดตั้งแอพ
- [ ] แสดงคำแนะนำติดตั้ง (ถ้ายังไม่ติดตั้ง)
- [ ] ติดตั้งได้บน Android/iOS/Desktop
